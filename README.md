# ShelfSmart
SmartShelf is an IoT-enhanced Android app that helps students and librarians search, track, and manage library books effortlessly.SmartShelf is a modern Android application designed to simplify library book tracking using IoT integration. Students can search books, view availability, and monitor their borrowed items, while librarians can manage inventory and track shelf status in real time. With a clean interface and Firebase backend, SmartShelf provides a fast, reliable, and user-friendly library experience.
